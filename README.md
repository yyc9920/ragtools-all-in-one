# ragtools-all-in-one
all in one rag tools for personal needs

## Setup Virtual ENV

```bash
git clone https://github.com/yyc9920/ragtools-all-in-one.git
python -m venv ragtools-all-in-one
cd ragtools-all-in-one

# Mac or linux
source ./bin/activate

# Windows
Scripts\activate

pip install -r requirements
```
